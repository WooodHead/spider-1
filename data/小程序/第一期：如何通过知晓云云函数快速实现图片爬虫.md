本文会跟大家介绍，如何在本地使用知晓云云函数 API 与 cheerio 开发一个图片爬虫，并使用 webpack 与 mincloud 将代码打包并上传部署到知晓云。

技术栈：

*  代码打包工具：webpack@4.20.2
*  部署工具：mincloud@1.0.2
*  html 解析：cheerio@1.0.0-rc.2
*  其他：知晓云 Node SDK（云函数内置）

*  将下载下来的图片上传到知晓云
